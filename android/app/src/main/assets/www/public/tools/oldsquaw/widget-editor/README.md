

# Oldsquaw-Widget-Editor

## 介绍
Oldsquaw-Widget-Editor 是一个用于编辑小部件的工具。它提供了一个直观的用户界面，允许用户轻松地创建和修改各种小部件。本项目旨在为开发者提供一个高效、便捷的小部件编辑解决方案。

## 软件架构
该项目采用模块化设计，主要组件包括：
- **HTML结构**：定义了编辑器的基本布局和用户界面元素。
- **CSS样式**：负责编辑器的外观和交互效果。
- **JavaScript逻辑**：实现了编辑器的功能和交互逻辑。

## 安装教程
1. 克隆仓库到本地：
   ```bash
   git clone https://gitee.com/oldsquaw/oldsquaw-widget-editor.git
   ```
2. 进入项目目录：
   ```bash
   cd oldsquaw-widget-editor
   ```
3. 安装依赖（如果项目使用了包管理器）：
   ```bash
   npm install
   ```
4. 启动开发服务器（如果适用）：
   ```bash
   npm start
   ```

## 使用说明
1. 打开编辑器页面后，您可以在 `.editor-container` 区域进行小部件的编辑。
2. 使用 `#editor` 区域进行主要内容编辑。
3. 工具按钮位于 `.button-container` 区域，支持下拉菜单功能（`.dropdown` 和 `.dropdown-content`）。
