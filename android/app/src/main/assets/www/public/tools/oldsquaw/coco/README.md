
# CoCo-Oldsquaw 编辑器使用说明  

## 项目简介  
CoCo-Oldsquaw 是基于 CoCo 编辑器的扩展优化项目，旨在通过功能增强与体验升级，帮助用户更高效地使用编辑器进行创作。项目支持**直接部署后即用**，无需复杂配置，适合开发者、设计师及内容创作者快速搭建个性化编辑环境。  


## 快速使用指南  
**部署方式**：  
1. 克隆或下载本仓库代码到本地；  
2. 直接运行 HTML 文件（如 `index.html`）或部署至 Web 服务器，即可启动编辑器。  


## 核心功能列表  

| **功能分类**         | **详细说明**                                                                 |  
|----------------------|-----------------------------------------------------------------------------|  
| **界面美化**         | 1. 全局圆角设计，视觉风格更柔和，操作体验更舒适。<br>2. 支持**黑白双色模式**，自动优化 DarkMode 积木配色，适配昼夜使用场景。 |  
| **屏幕与布局优化**   | 集成**超宽屏幕适配功能**，搭配界面美化，横屏作品创作更流畅，适合宽屏设备或横版内容设计。 |  
| **多端交互增强**     | 1. 添加 **CoCoClick 功能**，手机用户可直接点击添加控件，提升移动端操作便利性。<br>2. 电脑端支持**点击+拖拽双模式添加控件**，满足不同操作习惯。 |  
| **协作与扩展能力**   | 1. **高级协作功能**：生成专属协作链接，一键解决自定义控件无法协作的问题，支持多人实时协同编辑。<br>2. 搭载 **CCWMPro 1.3 控件商城**：内置 30+ 精美控件，支持批量导入，大幅提升创作效率。<br>3. **CoCo Quick Import**：支持拖拽方式快速导入本地控件，简化资源管理流程。 |  
| **审核与 AI 辅助**   | 1. 通过 **CCP-Player** 实现自定义控件绕审核功能，保障内容发布灵活性。<br>2. 全新集成 **DeepSeekV3 0324 版本 AI CoCo 鸭**，提供智能提示与创作辅助，助力高效完成作品。 |
