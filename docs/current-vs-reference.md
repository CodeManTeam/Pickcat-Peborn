# Pickcat 当前原型与截图对比

## 本地资料

- 截图目录：`research_downloads/screenshots`
- 截图索引：`research_downloads/screenshot-index.json`
- 主要参考帖：
  - `403032` 使用教程
  - `419825` 使用技巧
- API 线索：
  - `docs/api-map.md`
  - `research/pypi/aumiao`

## 已接近原版

- 底部 5 Tab 结构已经齐全：首页、喵圈、发布、消息、我的
- 首页顶部、搜索、登录、帖子详情、用户页、作品阅览页都已具备可运行骨架
- 帖子流、板块、作品发现、推荐用户已接入真实社区残留接口
- 帖子作者、作品作者、消息发送者、评论作者已补本地用户页跳转
- 推荐 / 发现 / 关注 3 个 feed 的底部状态文案已分开处理

## 仍与截图有差异

- 首页卡片密度和原版 Pickcat 仍不完全一致
- 发布页目前偏原型，未完全还原原版表单布局与素材排布
- 个人页仍以社区接口近似映射，不是原 Pickcat 的完整状态页
- 点赞接口目前先以论坛回帖点赞为主，帖子/作品点赞还没统一打通
- 页面切换已加过渡，但还不是原生 App 级别的手势/层级动画

## 已确认的作品类型

- 首页推荐位多为视频预览，不是完整交互播放器
- 站内已确认至少存在两类作品来源：
  - `Kitten`
  - `NEMO`
- 作品详情外层框架接近一致，但内部 player 来源不同

## 下一步优先级

1. 继续抓社区内更多 Pickcat 历史截图并补充索引
2. 细化首页和发布页布局，使其更接近 `403032` / `419825`
3. 继续补用户页、作品页、消息页的真实接口映射
4. 追更多编辑器作品类型，补不同 player 的兼容展示
