关于 Pickcat Reborn

Pickcat Reborn 是编程猫社区的前端复刻计划，使用纯 HTML/CSS/JS 构建，以手机端体验优先，同时兼容桌面端。项目还原了社区核心功能：首页动态推荐、作品浏览、喵圈、搜索、个人主页、发布创作等，并接入社区 API 实现真实登录和数据交互。

项目致谢

本项目的创作工具模块接入了 Oldsquaw 系列作品（BetterNemo / BetterNemoPC / CoCo Pro / KN-Oldsquaw / 控件编辑器），以及 HHCL233 的 BetterNemoPC 网页适配版。感谢 Oldsquaw 和社区开发者们的贡献。

感谢编程猫社区的原作者们创造了这个平台，以及所有在社区中分享作品、教程和灵感的创作者。
